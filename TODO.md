## Konversi ke APK Android (Flutter)

- [x] Cek struktur project Flutter
- [ ] Pastikan Flutter sudah terpasang dan bisa build
- [ ] Jalankan `flutter pub get` di `flutter_app/`
- [ ] Jalankan `flutter build apk --release` di `flutter_app/`
- [ ] Cek file APK hasil build di `flutter_app/build/app/outputs/flutter-apk/`
- [ ] (Opsional) Buat keystore & signing release (bukan debug)
